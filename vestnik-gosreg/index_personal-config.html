Реквизиты учетной записи

Официальные лица

Имеющие право подписи (заполнять форму в родительнои падеже). Данные этого раздела требуются для заполнения договоров.

Сменить пароль

В данном разделе вы можете изменить свой текущий пароль. Для изменения пароля необходимо знать действующий.

Электронная подпись

Если привязать электронную подпись к учетной записи личного кабинета, это позволит проходить авторизацию в личном кабинете по электронной подписи без ввода пароля.

Банковские реквизиты

Данные реквизиты могут использоваться для выставления счетов, оформления договоров и сопутствующих отгрузочных документов. Количество добавлямых банковских реквизитов не ограничено.
